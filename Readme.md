# Pomodoro Timer

Um cronômetro Pomodoro moderno, personalizável e focado em produtividade, desenvolvido para auxiliar nos estudos e no gerenciamento de tempo diário. Este projeto demonstra conceitos de **React**, **TypeScript**, **Tailwind CSS** e gerenciamento de estado em tempo real.

## 🌐 TESTE O PROJETO AQUI: https://pomodoro.ferreiradev.com/

## 📋 Características

### Funcionalidades Implementadas

- ✅ **Ciclos Personalizáveis**: Configure tempos de foco, descanso curto e descanso longo.
- ✅ **Notificações Sonoras**: Alertas sonoros ao final de cada ciclo para manter o foco. 
- ✅ **Interface Dark Mode**: Design escuro nativo para reduzir o cansaço visual.
- ✅ **Progress Tracking**: Visualização clara do progresso atual do ciclo.
- ✅ **Design Responsivo**: Experiência otimizada para mobile, tablet e desktop.
- ✅ **Persistência de Configurações**: Suas preferências de tempo salvas localmente.
- ✅ **Controle de Estado**: Pausa, reinício e pulo de ciclos com um clique.

## 🚀 Como Executar

### Pré-requisitos                                 

- Node.js 18+ instalado
- npm ou pnpm como gerenciador de pacotes

### Instalação

```bash
# Clonar o repositório
git clone https://github.com/LucasFerreiraProgramador/pomodoro-timer.git
cd pomodoro-timer

# Instalar dependências
npm install
# ou
pnpm install
```

### Desenvolvimento

```bash
# Iniciar servidor de desenvolvimento
npm run dev
# ou
pnpm dev
```

### Build para Produção

```bash
# Compilar para produção
npm run build
# ou
pnpm build

# Iniciar servidor de produção
npm start
# ou
pnpm start
```

## 🎨 Design e Estilo

### Paleta de Cores (Dark Theme)

- **Fundo**: `#0f172a` (Slate Dark)
- **Destaque Foco**: `#ef4444` (Red 500)
- **Destaque Descanso**: `#10b981` (Emerald 500)
- **Texto**: `#f8fafc` (Slate 50)
- **Texto Secundário**: `#94a3b8` (Slate 400)

### Componentes de UI

- **Cards**: Bordas suaves e sombras sutis para foco no cronômetro.
- **Botões**: Feedback visual no hover e estados ativos.
- **Inputs**: Estilizados para fácil ajuste de minutos e segundos.
- **Navegação**: Menu simples para alternar entre tipos de descanso.

## 💡 Conceitos de Aprendizado

Este projeto demonstra a aplicação de:

- ✅ **React Hooks**: Uso extensivo de `useState`, `useEffect` e `useRef` para precisão do tempo.
- ✅ **TypeScript**: Tipagem rigorosa para garantir a integridade dos dados de configuração.
- ✅ **Tailwind CSS**: Estilização rápida e responsiva com classes utilitárias.
- ✅ **Audio API**: Integração de sons nativos do navegador para alertas.
- ✅ **Web Storage**: Persistência de dados com `localStorage`.
